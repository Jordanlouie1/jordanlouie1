## How to deploy / demo
- Static sites (e.g., `twinvivo.ai`) — deploy easily to GitHub Pages, Netlify, or Vercel. Add a `netlify.toml` or `vercel.json` to speed setup.
- Node/TypeScript apps — build with `npm run build` then deploy the `dist/` to your host of choice. Include `Procfile` for Heroku.
- Python/ML demos — provide a minimal `requirements.txt` and a small launch script like `app.py` with an optional `Procfile`.
